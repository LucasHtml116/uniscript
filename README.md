# uniscript
A new mini programing language to html for improve frontend for starters
it doesn't needs node or any provider, just keep editor.html, bundle.js and the icons and it will work!
open the directory project/ to see the original TS files

enter on
```link
https://uniscript.neocities.org
```
code example:
```uniscript
<!uniscript start!>
<!uniscript main, uniscript, content = <

config execution = {start: "code"}
>>;

<!uniscript code, html, content = <
<h1>Hi uniscript!</h1>
>>;
<!uniscript end!>
```
