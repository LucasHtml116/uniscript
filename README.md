# uniscript
A new mini programing language
enter on
```link
https://uniscript.neocities.org
```
code example:
```uniscript
<!uniscript start!>
<!uniscript main, uniscript, content = <

config execution = {start: "code"}
>>;

<!uniscript code, html, content = <
<h1>Hi uniscript!</h1>
>>;
<!uniscript end!>
```
